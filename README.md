# LLM 기반 자율형 AI 에이전트 개발자 양성 과정

- 초보자분들이 AI 에이전트의 핵심 인프라와 제어 로직을 쉽게 이해하고 실습해 볼 수 있도록 **[기초 파이썬 프로그래밍과 FastAPI 기본 서버 구축]** 및 **[LLM 핵심 프롬프트 엔지니어링]** 단원을 중심으로 구성한 강의 노트입니다.

## 1. Python 기초 및 FastAPI 서버 구축

AI 에이전트가 사용자의 요청을 수신하고 데이터를 처리하기 위해서는 웹 서버 아키텍처가 필수적입니다. 
파이썬 기초 문법을 다지고, 현대적인 비동기 웹 프레임워크인 **FastAPI**를 이용해 첫 API 엔드포인트를 구축을 알아보겠습니다.

## 2. LLM 아키텍처 및 고급 프롬프트 엔지니어링

LLM(대형 언어 모델)이 똑똑하게 판단하고 논리적으로 행동하게 하려면 체계적인 프롬프트 설계가 필요합니다. 
모델의 답변 정확도를 극대화하고 자율성을 부여하는 핵심 기법들을 배워보겠습니다.

### 단원 요약 및 에이전트 개발자 체크리스트

- **독립된 환경 준비**: `venv` 등을 활용해 독립된 가상환경을 구축하고 VS Code에서 코드를 작성합니다.
- **소통 인프라 마련**: 외부 시스템이 우리 에이전트와 통신할 수 있도록 **FastAPI 엔드포인트**를 확장해 나갑니다.
- **지능적인 제어**: LLM의 할루시네이션(거짓 답변)을 억제하기 위해 **CoT 기법**과 **출력 구조화** 프롬프트를 기본으로 장착해 줍니다.
